# SILAB BE

## Getting started

**Clone**
```
git clone https://github.com/tantowish/silab-be.git
cd silab-be
```

**Setup Environment**
```
cp .env.example .env
```

**Install local dependencies**
```
composer install
```

**Generate an app encryption key**
```
php artisan key:generate
```

**Migrate the database**
```
php artisan migrate
php artisan db:seed // optional if seed available
```

**Running Service**

```
php artisan serve 
```
